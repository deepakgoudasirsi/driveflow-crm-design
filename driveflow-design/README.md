# DriveFlow — Lead Management CRM (Design Deliverable)

Interview-level desktop UI design for DeltaX Product Associate assignment.

## What's included

| File | Description |
|------|-------------|
| `index.html` | Interactive 1440px prototype — all 4 screens |
| `styles.css` | Design system tokens (8pt grid, Inter, #2563EB) |
| `app.js` | Screen navigation & micro-interactions |
| `UX-DECISIONS.md` | Full rationale for every UX choice |
| `FIGMA-SPEC.md` | Auto Layout component & frame specs for Figma |

## Screens

1. **Lead Listing** — Filterable table, bulk actions, lead scoring
2. **Lead Details** — Timeline, deal info, tasks, AI insight
3. **Pipeline (Kanban)** — Drag-ready board with stage values
4. **Dashboard** — KPIs, funnel, source mix, activity charts

## View the prototype

```bash
open index.html
```

Use the **bottom frame switcher** or **sidebar navigation** to move between screens. Click **Priya Sharma** to jump from Listing/Kanban → Details.

## Design specs

- **Width:** 1440px desktop
- **Font:** Inter
- **Primary:** #2563EB
- **Spacing:** 8pt system
- **Style:** HubSpot + Linear + Attio

## Figma note

This deliverable is a **high-fidelity interactive prototype** plus Figma recreation specs. Native `.fig` files require Figma Desktop — use `FIGMA-SPEC.md` to rebuild with Auto Layout, or import screenshots as reference layers.
